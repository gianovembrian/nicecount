Dokumentasi Pengaturan Deteksi

Catatan penggunaan
Perubahan pada halaman ini hanya berlaku untuk analisis baru. Jika video sudah selesai dianalisis, hapus log deteksi atau jalankan analisis ulang agar hasil memakai pengaturan terbaru.
Parameter Fungsi Jika dinaikkan Jika diturunkan
Global Confidence Ambang confidence dasar dari model YOLO sebelum hasil deteksi diproses lebih lanjut. False positive berkurang, tetapi motor kecil bisa lebih sering tidak terdeteksi. Recall naik, tetapi noise dan deteksi palsu bisa bertambah.
Motorcycle Min Confidence Ambang minimum khusus untuk deteksi sepeda motor. Deteksi motor lebih selektif, tetapi motor jauh/kecil bisa hilang. Motor kecil lebih mudah masuk, tetapi risiko objek bukan motor ikut terdeteksi naik.
Car / Bus / Truck Min Confidence Ambang minimum khusus untuk mobil, bus, dan truk setelah threshold global. Hasil lebih bersih untuk class terkait, tetapi kendaraan ambigu bisa terbuang. Class terkait lebih mudah muncul, tetapi salah klasifikasi bisa meningkat.
IOU Threshold Mengatur seberapa agresif NMS membuang box yang saling overlap. Lebih banyak box overlap dipertahankan, berguna untuk motor yang berdekatan. Box overlap lebih cepat dibuang, tetapi motor dekat kendaraan lain bisa ikut hilang.
Frame Stride Mengatur jarak frame yang dianalisis. Nilai 1 berarti sampling paling rapat. Proses lebih cepat, tetapi motor cepat lebih mudah terlewat. Akurasi gerakan lebih baik. Untuk motor, disarankan tetap 1.
Target Analysis FPS Target jumlah frame per detik yang diproses untuk analisis. Motor cepat lebih stabil terdeteksi, tetapi proses analisis lebih berat. Proses lebih ringan, tetapi objek kecil/cepat lebih mudah skip.
Working Max Width Lebar maksimal frame kerja yang masuk ke pipeline inferensi. Detail objek meningkat, motor kecil lebih mudah dikenali, tetapi proses lebih lambat. Analisis lebih cepat, tetapi detail motor kecil bisa turun.
Preview FPS Frekuensi update preview saat proses analisis berjalan. Preview terlihat lebih sering diperbarui, tetapi penggunaan I/O sedikit naik. Preview lebih ringan, tidak banyak mempengaruhi akurasi deteksi.
Preview Max Width Lebar maksimal gambar preview yang dikirim ke UI. Preview lebih detail, ukuran file preview lebih besar. Preview lebih ringan, detail tampilan berkurang.
Preview JPEG Quality Kualitas kompresi JPEG untuk gambar preview. Preview lebih jelas, tetapi ukuran file lebih besar. Preview lebih kecil dan ringan, tetapi kualitas visual turun.
Rekomendasi saat banyak motor tidak terdeteksi
Pertahankan Frame Stride di 1, naikkan Target Analysis FPS ke 20 atau 25, coba Working Max Width 1920, dan naikkan IOU Threshold ke sekitar 0.55 jika banyak motor berdekatan.
Batasan model
Jika raw output YOLO tetap tidak mengeluarkan class motorcycle, perubahan setting tidak akan cukup. Solusi paling kuat adalah fine-tuning model dengan dataset CCTV lokal yang berisi motor Indonesia, sudut kamera tinggi, objek kecil, dan kondisi jalan nyata.
